# ⛏️ Digger 2D

Deskop game developed with Pygame.

**Controls:**

- Move: WASD
- Place block: K
- Delete block: L
- Select block: 1-6
- Reload game: R
- Chat: T

# Demo

You need `python3.12` installed on your machine

```bash
./demo.sh
```
